Práctica 1

Para esta primera práctica he desarrollado un CV utilizando HTML y CSS, el cual se puede compartir online. Es un CV de un personaje de una de mis sagas favoritas, "Los Juegos del Hambre".
El CV se adaptarse bien a dispositivos móviles. También, esta compuesto de una página principal que se debe llamar index.html. Y una página a la que se accede desde la principal con más información llamada about.html.
