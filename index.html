Práctica 1

Para esta primera práctica he desarrollado un CV utilizando HTML y CSS, el cual se puede compartir online. Es un CV de un personaje de una de mis sagas favoritas, "Los Juegos del Hambre".
El CV se debe adaptar bien a dispositivos móviles. También, esta compuesto de una página principal que se debe llamar index.html. Y una página a la que se accede desde la principal con más información llamada about.html.

Práctica 2

La segunda práctica consiste en la programación de un juego mediante HTML, CSS y JavaScript. El juego consiste en adivinar una clave secreta que se ha creado de forma aleatoria con números del 0 al 9, en el menor tiempo posible.
El juego contiene unos botones que se usan para seleccionar el valor que se quiere comprobar en la clave secreta. También, el juego está provisto de tres botones adicionales que se usan para iniciar el cronómetro, parar el tiempo y para reiniciar el juego.

Práctica 3

Práctica 4

Espero que tengas buena memoria... es hora de jugara a Memory!!
Desde un menú podrás ajustar tus preferencias de juego, como el tamaño del tablero, el tema de las cartas o el modo de juego.
Una vez empiece el juego, se trata de encontrar todas las parejas y cumplir el objetivo del modo de juego que hayas elegido.

Práctica 5

Aplicación en javascript que simula el envío de paquetes a través de internet.
Se genera una red de nodos conectados de manera aleatoria. El número de nodos totales es elegido por el usuario y lo mismo con el nodo origen y el destino.
Una vez generada la red, se calcula la ruta más corta mediante Dijkstra. El reultado de la ruta queda marcada en el dibujo y se escribe.
